archanjOS – Sistema de Consulta e Wishlist de Jogos
Status: Em desenvolvimento

Tecnologias utilizadas:
Java (Swing),
MySQL,
JDBC,
Git / GitHub,
MySQL Workbench,
NetBeans.

Time de Desenvolvedores
Bruno Archanjo


Objetivo do Software
O sistema archanjOS tem como objetivo permitir que o usuário consulte rapidamente jogos disponíveis em diversas plataformas, visualize detalhes como preço, descrição e imagem, e adicione os títulos desejados a uma lista de desejos armazenada no banco de dados.

Funcionalidades
Tela de Login:
Autenticação simples (usuário: admin).


Tela de Busca:
Campo para digitar o nome do jogo.


Seleção de plataforma (Steam, Xbox, PlayStation, Switch).


Botão Search.


Navegação para a wishlist.


Tela de Detalhes:
Exibição de nome, preço, imagem, descrição e plataforma.
Botão para adicionar o jogo à wishlist.


Tela de Wishlist:
Lista de jogos cadastrados no banco.
Nome, plataforma e preço atual.
Navegação de volta para busca.


Integração com banco MySQL:
Tabela jogos.
Tabela wishlist.


Conexão via JDBC usando ConnectionFactory.

Versionamento:
commits, pushs e pulls demonstrando evolução do projeto.
